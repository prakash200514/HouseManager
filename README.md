# House Rental System


## 📔 Description
House Rental System is a C2C property rental website that connects property owners directly with renters. This comprehensive web application features user and booking management, detailed profiles, and robust property search functionality, ensuring a seamless, user-friendly experience for all.<br/><br/>

## 💡 Technology
- Frontend Language: HTML, CSS, JavaScript
- Backend Language: PHP
- Database: MySQL<br/><br/>
